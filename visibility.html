Visibility

 Top  Previous  Next

Some information managed in the Catalog Service is marked with a visibility attribute. This visibility attribute is a hint to other systems using the Catalog Service to dynamically adapt user interfaces according to the role the accessing user has.

The Catalog Service does not enforce this visibility in any way, it is up to the systems using the Catalog Service to implement the visibility accordingly.

Visibility

Description

Everyone

An element with this visibility shall be shown to every user of the HsM system.

Maintenance

An element with this visibility shall only be shown to users in the HsM system that care about maintenance of devices and the HsM System.

Administrator

An element with visibility shall only be shown to users who administrate the HsM instance of a HsM OEM Partner.

Manufacturer

An element with this visibility shall only be shown to users to Developers only shown to the manufacturer of the devices or the HsM system.
