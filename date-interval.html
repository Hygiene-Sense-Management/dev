Date interval specification

 Top  Previous  Next

Date intervals are used to filter or query timestamped data that is within an given start date and end date. Since HsM deals with time-series a lot the Client Service provides versatile ways of specifying date intervals.

Static date interval specifications contain a fixed start date and end date. Adaptive date intervals specifications are defined with offsets - the actual date interval is then calculated by applying the defined offsets to the current date and time when the date interval is accessed.

Both types of date intervals specifications also allow to specify if the start date and end date are included in the date interval or not.
