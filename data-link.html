Data link capability

 Top  Previous  Next

The HsM system can also manage pure analog dispensers (with no data link to the HsM Backend), this device filter term type can be used to filter devices by their capability to send data to the <%BACKEND%.

Note this term type will not filter for devices that actually send data to the HsM Backend but only filters for their theoretical ability to send data.

Settings

Setting

Description

Data link capability

Can send data: Filters for devices that can send data to the HsM Backend.

Cannot send data: Filters for devices that cannot send data to the HsM Backend

Include compound members

If a device is a compound device and has a compound member that can send data to the HsM Backend these devices will pass the filter if this option is set.
