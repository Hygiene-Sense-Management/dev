Location tree filter

Filters for devices that are installed at specific locations with in the location tree of customers.

Settings

Setting

Description

Locations

Devices that are assigned to any of the given locations or any of its sub-locations will pass the filter.

Table 11: Settings of this device term type

Validity constraints

•At least one location must be set (hard validity constraint): If no location is set this filter term will not match any device.

•All locations must be valid locations (soft validity constraints): If a location does not exist it will be ignored during execution.
