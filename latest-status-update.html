Filters for devices that sent a resource instance value (for any resource instance) within the configured time span.

Settings

Setting

Description

Time span

The filter will be passed by all devices who have reported a value for any resource instance later than CurrentDateAndTime - TimeSpan .

CurrentDateAndTime is evaluated at the time when the filter is executed.

Table 9: Settings of this device term type

Validity constraints

Time span greater 60s (hard validity constraint): The length of the given time span must be greater than 60 seconds.
