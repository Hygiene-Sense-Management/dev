Compound device status

 Top  Previous  Next

Compound member devices are part of the base set of devices to which device filters are applied, i.e. in the base set of devices to which the device filter is applied to includes compound member devices as well as compound devices

The Compound device status term type can be used to restrict the result based on the actual links between devices.

Settings

Setting

Description

Compound status

One of the following options:

•Is compound member device: Will filter for devices that are linked to other devices

•Is compound device: Filters for devices that include other devices

•Is not compound device: Filters for devices that have no included devices

•Is not compound member device: Filters for devices that are not included in another device.

Table 3: Settings of this device term type

Validity constraints

This term type has no validity constraints.
